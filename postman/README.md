# Postman Assets

Add exported files from Postman to this folder.

Expected files:
- Multi-Tenant-Payment-Gateway.postman_collection.json
- Multi-Tenant-Payment-Gateway.local.postman_environment.json

Export guidance:
1. Open collection in Postman.
2. Use Export and choose Collection v2.1 JSON.
3. Export local environment variables as JSON.
4. Save both files in this folder.
